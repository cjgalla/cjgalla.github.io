Peer Review 1

Gerasymenko, Bogdan

Evaluation

  • Contrast: Pretty good contrast overall, text is readable against background colors only thing I would say is there is 3 or 4 different text colors used and that there are some colors that are getting contrast errors.
  • Repetition: Fairly consistent design, the use of sections and borders is proper it helps keep things apart, however there are things like images not showing up, or being different sizes in their respective elements that I would say there could be some work done on those.
  • Alignment: Elements are aligned well, although some spacing could be refined for even better visual flow, overall it is readable, I would just worry about small screen sizes trying to keep up.
  • Proximity: Related items are properly grouped, only real thing for this I have is the about page is after all the real meat of the website on the nav bar.
  • Header/Main/Footer: Proper structure present. Site has a clear header, main content, and footer.
  • Navigation: Navigation bar is present and working properly at all times.
  • Tagline/Slogan: No tagline present, although not imperative it would be nice to have one.
  • File/Folder Naming: No uppercase letters or spaces in file or folder names, looking good!
  • Validation: Site has no actual errors, only false ones on things accumulus doesnt recognize, there are some interesting warnings as there are 2 h1 and h2 tags called in cs and 4 h3, but after reviewing they seem to be for different classes.
  • Responsiveness: Site is mostly responsive, but the site cant fit onto screen at one time and there is no responsiveness for smaller screen sizes.
  • Additional Comments:
    • Overall, the site looks professional and cohesive, and it is navagatable, I only really have knitpicky things for improvement to just polish things over.
    • Could benefit from a little more responsiveness for smaller screen sizes, and there could be a little bit of touching up on some image sizes to make things cohesive and repetitive.
    • Otherwise though the website is fully functional so the only real issues are visual!
